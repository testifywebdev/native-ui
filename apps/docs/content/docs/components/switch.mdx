## API Reference

### Switch

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `boolean` | — | **Required.** Current on/off state. |
| `onValueChange` | `(value: boolean) => void` | — | **Required.** Callback when state changes. |
| `disabled` | `boolean` | `false` | Prevents toggling and reduces opacity to 50%. |
| `accessibilityLabel` | `string` | — | Screen reader label — highly recommended. |
| `accessibilityHint` | `string` | — | Additional screen reader context. |

> **Dimensions:** track is 44×24px, thumb is 20×20px. These are fixed constants to match standard iOS/Android switch proportions. Customize via the source if needed.
