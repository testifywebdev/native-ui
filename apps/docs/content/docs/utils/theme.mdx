## Overview

All NativeUI components consume a central `THEME` object and `useTheme` hook from `@/lib/theme`. The theme defines four tokens categories:

| Category | Description |
|----------|-------------|
| `colors` | Semantic color palette (background, foreground, primary, muted, …) |
| `spacing` | Consistent spacing scale from `xs` (4) to `3xl` (32) |
| `radius` | Border-radius scale from `sm` (4) to `full` (9999) |
| `typography` | Font-size scale from `xs` (11) to `2xl` (24) |

## Setup

Wrap your root layout with `ThemeProvider`:

```tsx title="app/_layout.tsx"
import { ThemeProvider } from '@/lib/theme';

export default function RootLayout({ children }) {
  return (
    <ThemeProvider>
      {children}
    </ThemeProvider>
  );
}
```

## Usage in components

```tsx
import { THEME, useTheme } from '@/lib/theme';

function MyComponent() {
  const { theme } = useTheme();
  const { colors, spacing, radius, typography } = THEME[theme];

  return (
    <View style={{
      backgroundColor: colors.card,
      padding: spacing.lg,
      borderRadius: radius.lg,
    }}>
      <Text style={{ fontSize: typography.base, color: colors.foreground }}>
        Hello!
      </Text>
    </View>
  );
}
```

## Changing the theme

```tsx
import { useTheme } from '@/lib/theme';

function ThemeToggle() {
  const { theme, setTheme } = useTheme();

  return (
    <Button
      title={theme === 'dark' ? 'Switch to Light' : 'Switch to Dark'}
      onPress={() => setTheme(theme === 'dark' ? 'light' : 'dark')}
    />
  );
}
```

## API Reference

### THEME

```ts
const THEME: Record<"light" | "dark" | "unspecified", ThemeConfig>
```

A static object containing the full theme config for each color scheme. Access it as `THEME[theme]` where `theme` is the current value from `useTheme()`.

### ThemeProvider

```tsx
<ThemeProvider>
  {children}
</ThemeProvider>
```

Reads the device color scheme via `useColorScheme()` and sets the initial theme. Provides `theme` and `setTheme` via context.

### useTheme

```ts
const { theme, setTheme } = useTheme();
```

| Return value | Type | Description |
|--------------|------|-------------|
| `theme` | `"light" \| "dark" \| "unspecified"` | Current active theme. |
| `setTheme` | `Dispatch<SetStateAction<Theme>>` | Update the active theme. |
